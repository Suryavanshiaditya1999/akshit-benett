Exercise 1: Age Categorizer

```
# Write a program that:
# 1. Asks user for their age
# 2. Categorizes them based on age:
#    - Under 13: Child
#    - 13-19: Teenager
#    - 20-60: Adult
#    - Above 60: Senior Citizen
# 3. Prints their category

# Example:
# Enter your age: 25
# Output: You are an Adult
```

```

Exercise 3: List Operations

```

# 1. Create a list of 10 numbers
# 2. Find and print:
#    - Largest number
#    - Smallest number
#    - Sum of all numbers
#    - Average of numbers
#    - Numbers greater than the average

# Example Output:
# Original list: [10, 5, 8, 12, 3, 15, 7, 20, 1, 6]
# Largest number: 20
